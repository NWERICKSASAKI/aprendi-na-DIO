Estrutura Básica do HTML

HTML é a linguagem de marcação básica para a criação de páginas da web.
É composta por tags que abrem com < e fecham com >. O contéudo fica inteiramente dentro da tag <html>, dentro deles há duas tags principais:

  1. <head> - O conteúdo contido no head carrega antes da página aparecer para o usuário
  2. <body> - O conteúdo contido no body é o que aparece ao usuário

Atributos das Tags

Atributos provêm informações adicionais aos elementos HTML.
Eles são sempre especificados na abertura das tags e comumente são inseridos em pares como: name="value"

A seguir alguns exemplos de atributos:

  • <strong id="titulo" - usado para identificação para ser manipulado posteriormente via javascript;
  • <strong style="color:blue" - usado para atribuir um style ao elemento, sobrescrevendo ao style global;
  • <strong class="titulo-principal" - usado para identificação e padronização para pré-definições em CSS;
  • <input type= - os atributos a seguir são exclusivos da tag input, na qual é usado para coletar entradas do usuário.
    • "text" - usado para a coleta de texto;
    • "number" - usado para a coleta de valor numéricos;
    • "color" - usado para a coleta de cor;
  • <img src="imagem.jpg" - usado para inserir imagem, pode ser um caminho interno ou url externo;
  • <img width="250" - usado para alterar (proporcionalmente) o tamanho da imagem, usando o valor da largura como parâmetro de definição;

Textos

As principais tags de textos são apresentadas a seguir:

  • <u> - Usado para deixar o texto sublinhado;
  • <i> - Usado para deixar o texto em itálico;
  • <b> - Usado para deixar o texto em negrito
  • <strong> - Também deixar em negrito porém mais semântico;
  • <h1> - Usado para título principal;
  • <h2> - Usado para título secundário, pode se usar h3, h4 e assim em diante para subtítulos;
  • <p> - Usado para declarar um parágrafo;
  • <q> - Usado para declarar citação curta;
  • <blockquote> - Usado para declarar citação longa;
  • <mark> - Usado para destacar como se fosse um marca-texto;
  • <sup> - Usado para sobrescrita, exemplo: ex;
  • <sub> - usado para subscrita, exemplo: H2O.

Listas

Lista Ordenada

Ordened list, ou listas ordenadas, são declaradas com <ol> e dentro dele são inserido os seus list itens itens, ou itens da listas, <li>
Exemplo:

<ol>
    <li>Um item</li>
    <li>Mais um item</li>
    <li>Outro item</li>
</ol>

  1. Um item
  2. Mais um item
  3. Outro item

Lista Não Ordenada

Unordened list, ou lista não ordenada, são declaradas com <ul> e dentro dele são inserido os seus list itens itens, ou itens da listas, <li>
Exemplo:

<ul>
    <li>Um item</li>
    <li>Mais um item</li>
    <li>Outro item</li>
</ul>

  • Um item
  • Mais um item
  • Outro item

Links

Conhecidos também por anchor, ou âncora, textos que, quando clicados, navegam através da página, para outra página do site ou para fora do site. Exemplo:

  • <a href="#nome-do-id" - navega pela própria página para o elemento com o id declarado
  • <a href="contatos.html" - navega para outra página do mesmo site ou projeto
  • <a href="http://dio.me" - navega para uma página externa

Além da aplicação do atributo href, temos outros atributos principais:

  • <a title="Clique aqui para saber mais" - Ao passar o mouse por cima do elemento aparecerá um tooltip
  • <a target=
    • "_self" - (padrão) usa aba atual para navegar ao novo endereço
    • "_blank" - abre uma nova aba para navegar ao novo endereço

Boa prática: "_self" é usado para navegar dentro do seu próprio site, enquanto o "_blank" é usado quando o link direciona para fora do seu site.

Formulário

<form> é usado para coletar informações do usuário, delimita (abraço) todos os demais elementos de formulários que serão apresentados posteriormente.
Primeiro vamos apresentar seus principais atributos:

  • <form name="signup" - atribuído para orientar o javascript;
  • <form action="http://meusite.com.br/signup" - endereço do servidor que as informações do formulário irão;
  • <form method=
    • "get" - envia as informações através da própria url
    • "post" - envia as informações encapsuladas na body
  • <form autocomplete="on" - caso "on" o usuário pode salvar as credenciais, assim como se voltar para a páginas as informações estarão pré-preenchidas
  • <form on-alguma-coisa - todas os atributos começados por "on" são eventos;
  • <form onsubmit="alert('formulario enviado');" - neste exemplo, quando dado o comando de enviar o formulário, executará o comando javascript descrito;
  • <form target - similar ao link <a>
