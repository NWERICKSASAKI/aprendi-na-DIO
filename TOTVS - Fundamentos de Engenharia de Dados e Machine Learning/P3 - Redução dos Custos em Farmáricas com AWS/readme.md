# Redução dos Custos em Farmáricas com AWS

Este projeto proposto pela DIO visa aplicar os conhecimentos adquiridos durante o curso de **Noções de Computação em Nuvem** na qual se baseia nos serviços e infraestruturas fornecidas pela Amazon Web Service (AWS).  

O desafio consiste em elaborar um relátorio-proposta técnica para a modernização de uma farmácia fictícia, indicando quais ferramentas e serviços da AWS pode ser oferecido e sua respectiva proposta.  

O **modelo base** do relatório se encontra disponível em <modelo-relatorio.md>  
A **versão final** de entrega pode ser visualizado em <relatorio.md>.
