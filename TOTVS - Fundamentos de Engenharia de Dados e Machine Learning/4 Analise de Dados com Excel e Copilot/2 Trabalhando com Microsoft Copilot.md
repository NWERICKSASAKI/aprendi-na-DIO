### 1.3 Cuidados com o Resultado

**Teste de realidade**: é perguntar algo pra IA que você já sabe a resposta.

`Quanto está o dólar hoje?`

Retorna sempre uma fonte, sempre confira nas fontes, isso é importante para saber que a IA não está inventando e ver se as informações estão atualizadas.

*Não use IA para fabricar dados!*, use para analisar dados existentes.

### 1.4 Analisando tabelas

Convém enviar o arquivo csv com os dados para extrair os dados na IA.  

`Leia o arquivo e retorna a soma dos valores de venda`

O copilot também retorna os códigos usados para validar o resultado.

Nunca faça perguntas que possa fabricar dados.  

E não envie dados sensíveis na IA.
