# Analisando Bases de Dados com Excel e Copilot

## 1 Copilot Overview

### 1.1 Introdução Microsoft 365 Copilot

Precisar assinar para ter acesso ao Copilot dentro dos programas do Microsoft Office, como Excel, Word, Power Point, etc

Assim que habilitado, aparece um botão do Copilot no menu `Página Inicial`, para ativá-lo é necessário ativar o *salvamento automático*.

Exemplo, abrindo uma planilha excel de vendas de tênis, como data de venda, modelo, preço, quantidade, cidade, gênero e idade do comprador.

No chat do Copilot:

```txt
Verifique se existe registro duplicados
e me avise como remover
```

Nesse caso retornou uma fórmula para acrescentar em uma nova coluna que retorna se a linha é *única* ou *duplicada*.

```txt
Qual a idade média do público masculino e feminino que compram esses modelos de tênis?
```

Ele retorna um texto acompanhado com um gráfico com a resposta solicitada.

### 1.2 Insight buttons

Há um botão chamado `Mostrar insights de dados` na qual retorna alguns dados interessantes, como:

- Quantidade de venda por data
- Datas com maior quantidade de vendas
- Etc

Além de ter um botão `Adicionar a uma nova planilha` que já retorna uma planilha dinâmica com um gráfico em uma nova aba.

Nos prompts, o ideal é fazer uma análise por vez.

### 1.3 Insights de formação e colunas agregadas

O Copilot pode também:

- Aplicar formatações condicionais com base no prompt, clicando em `Aplicar` após rever os comandos que ele sugere;
- Criar coluna com total de vendas, multiplicando preço de venda por quantidade, depois só clicar em `Inserir coluna`;

### 1.4 Recursos do Painel de Chat

O Chat do Copilot também tem um **botão de microfone**, que permite que enquanto você fala ele digita no campo de prompt.

Do lado tem um ícone de **Exibir Solicitações** nas quais tem prompts pré-prontos.
