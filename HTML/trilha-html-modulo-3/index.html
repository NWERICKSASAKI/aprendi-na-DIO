Introdução

Boas vindas,
Esta é uma página com fins de desenvolvimento de entrega de projeto/atividade do Desafio de Projeto: Recriando o Wikipedia com Layout Moderno

Sobre o Desafio

Este desafio consiste em treinar você com relação a estrutura do site, e também aplicar conhecimentos adiquiridos sobre semântica e acessibilidade.

Todo código-fonte desenvolvido durante o curso foi versionado no GitHub, no seguinte endereço:

https://github.com/digitalinnovationone/trilha-html-modulo-3

Resultados

A temática para as páginas modernas de wikipedia foi escolhido jogos populares da Nintendo.

Como resultado do projeto, foi desenvolvido estes conjuntos de páginas, na qual há uma âncora para ser navegado pelo própria página ao lado direito e enquanto ao lado esquerdo permite visitar outras páginas.
